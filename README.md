## 🧩 Overview

This is my **personal portfolio website** that highlights my background, education, and skills in **DevOps and Cloud Engineering**.  
It’s designed to be **lightweight, fully containerized, and Kubernetes-ready** — perfect for deploying anywhere.

✨ Features:
- 💎 Modern responsive design (pure HTML + CSS)
- ⚙️ Nginx as web server
- 🐳 Dockerized for easy deployment
- ☸️ Kubernetes manifests (`Deployment`, `Service`, `Namespace`, `Pod`)
- 🧭 Works perfectly on **Kind**, **Minikube**, or **Cloud Clusters**
- 📱 Optimized for mobile and desktop
